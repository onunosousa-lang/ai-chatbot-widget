Create Config File (15 min)

Use Claude Code or manually create config.clientname.json

Quick Method with Claude Code:
Just say: "Create config for client 'TechStartup' with primary color #7C3AED, company name 'TechStartup Inc', email hello@techstartup.com"

Or manually:

cd /path/to/ai-chatbot-widget
cp config.template.json config.clientname.json
# Edit the file with client details
git add config.clientname.json
git commit -m "Add [Client Name] config"
git push origin main
Important: Client ID must be lowercase with no spaces (e.g., "techstartup", "acmecorp")
3
Create OpenAI Assistant (20 min)

Set up the AI that will power the chatbot conversations.

OpenAI Assistant Setup:
Go to https://platform.openai.com/assistants
Click "Create"
Name: "[Client Company] Customer Support Bot"
Copy instructions from ASSISTANT_INSTRUCTIONS_FINAL.md
Update company name in instructions
Update services/pricing in instructions
Model: gpt-4-turbo-preview
Enable: Function calling
Add save_lead function (see below)
Click "Save"
COPY THE ASSISTANT ID (e.g., asst_abc123xyz)

save_lead Function:

{
  "name": "save_lead",
  "description": "Saves contact info when provided",
  "parameters": {
    "type": "object",
    "properties": {
      "name": {"type": "string", "description": "Full name"},
      "email": {"type": "string", "description": "Email address"},
      "phone": {"type": "string", "description": "Phone number"},
      "preferred_time": {"type": "string"},
      "notes": {"type": "string"}
    },
    "required": ["name", "email"]
  }
}

5
Add to Vercel (10 min)

Configure environment variables for this client.

Vercel Configuration:
Go to https://vercel.com/dashboard
Select project: ai-chatbot-widget
Go to Settings ‚Üí Environment Variables
Add CLIENT_CLIENTNAME_ASSISTANT_ID
Paste Assistant ID from Step 3
Add CLIENT_CLIENTNAME_WEBHOOK_URL
Paste webhook URL from Step 4
Click "Save"
Go to Deployments ‚Üí Redeploy
Wait 2-3 minutes for deployment
Important: Replace CLIENTNAME with the client ID in UPPERCASE.
Example: If clientId is "techstartup", use CLIENT_TECHSTARTUP_ASSISTANT_ID

Example:

CLIENT_TECHSTARTUP_ASSISTANT_ID = asst_abc123xyz
CLIENT_TECHSTARTUP_WEBHOOK_URL = https://hook.eu1.make.com/xyz123
